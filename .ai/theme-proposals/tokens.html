Fraunces, ambitiously.

Fraunces carries 95% of the type, across four optical sizes and three SOFT values. DM Sans handles labels and micro-typography only. The italic WONK glyphs (WONK 1) are reserved for accent words — nothing else earns them.

viii.§ 08 · Usage

Four principles.

The rules that keep the system from drifting. Break them deliberately; bend them slowly.

i.

Tokens, only.

Component CSS never references a hex value. If a new color feels necessary, it belongs in the token system first, the component second.

ii.

One accent per viewport.

At any scroll position, the eye should land on one accent phrase, one accent motif — not a scatter. Restraint is the whole design.

iii.

WONK for emphasis only.

The italic alternate glyphs are reserved for the word being emphasized. Using them for full paragraphs kills the effect and looks like novelty.

iv.

Dark mode is the same design.

Layout, type scale, motif placement, and rhythm do not change between modes. Only the color tokens swap. If a section falls apart in dark, the problem is the section, not the mode.
